HPC Provenance & Reproducibility Framework

This research introduces a novel, secure, and transparent data provenance framework designed to automatically capture metadata in HPC systems without requiring user intervention. The primary goal is to enable data versioning and traceability, which are critical for supporting reproducible scientific experiments in complex computing environments.

Key Features & Contributions:

  • Developed a low-overhead tracing system using eBPF to monitor file I/O and process behavior.
  • Evaluated on real-world scientific applications, demonstrating minimal performance impact.
  • Ensures comprehensive provenance capture for enhanced security and accountability.
  • Published at HiPC 2025 (under submission) and presented at CSCE'24 and ICCCN 2024.

Dynamic Load Balancing in AMReX

During a summer internship at Lawrence Berkeley National Laboratory (LBL), I researched efficiency and load balancing improvements for mesh-and-particle frameworks. This led to the co-development of a novel algorithm to address performance bottlenecks in large-scale simulations.

Key Features & Contributions:

  • Proposed the "Painter Partition Solver," a new algorithm based on a modified Space-Filling Curve (SFC).
  • Developed a solution compatible with 1D, 2D, and 3D domains.
  • Integrated the solver with probabilistic load balancing strategies.
  • Empirical benchmarking demonstrated improved performance over existing SFC and Knapsack algorithms.
  • Accepted for publication at PEARC 2025.

I/O Optimization via Request Merging

This project, also conducted at LBL, focused on I/O optimization strategies for scientific computing. The core innovation was a technique to significantly boost write throughput by intelligently managing small, asynchronous write operations.

Key Features & Contributions:

  • Developed a method to merge multiple small, out-of-order writes into larger, contiguous writes.
  • Implemented support for 3D scientific datasets, improving performance under varying workloads.
  • Benchmarked on NERSC's Cori supercomputer, showing significant performance gains.
  • Gained hands-on experience with asynchronous I/O job scheduling and performance analysis.
  • Published at the IPDPSW 2023 conference.
